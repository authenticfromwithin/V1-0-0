AFW Admin-Ready Scripts (Offline)
---------------------------------
These Node 22 scripts help you create and validate content locally.

Usage examples:
  node scripts/admin/devo-new.mjs 2025-09-01 "Title here" "Psalm 23"
  node scripts/admin/quotes-add.mjs faith "Text of quote" "Author"
  node scripts/admin/validate.mjs

All scripts write minimal logs and exit(1) on fatal schema errors.
